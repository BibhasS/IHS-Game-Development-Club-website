GET STARTED

Think of this page like a quest log. Do these 4 steps, and you’ll be fully set up to start labs and get graded. If you get stuck anywhere, ask in Discord (#assistance).

Step 1 Step 2 Step 3 Step 4 Finish
Start the Quest

Join the Discord

Goal: get help + announcements + submit labs

Discord is where everything happens: questions, updates, grading threads, and the leaderboard.

  • Join the server
  • Say hi (so we know you’re in)
  • Ask for help anytime
Open Discord

Install + Learn the Godot Editor

Goal: understand the basics fast

This slideshow is the quickest way to understand the Godot IDE without getting overwhelmed.

  • Open the “Intro to Godot” slideshow
  • Follow the setup steps
  • Skim once, then re-check while installing
Intro to Godot Slides

Set up a Script Editor (C#)

Goal: autocomplete + debugging + fewer errors

Do not use Godot’s built-in script editor for C#. Use an IDE so it catches mistakes instantly.

  • Recommended: JetBrains Rider
  • Backup option: VS Code (if storage/performance is a problem)
Rider Setup + Conventions VS Code Setup (optional)

Do Lab 1 + Submit It

Goal: your first graded submission

Your first lab is at the end of the Intro to Godot slideshow: “Easy Artwork Printing.”

  • Complete the lab
  • Zip the project folder
  • Post it in #share-lab as: FirstName Lab 1

You’ll get graded in a private thread under #grading. You can resubmit as many times as you want based on feedback.

Ask for Help in Discord

✅ QUEST COMPLETE

Once you’ve done Steps 1–4, you’re officially set up. From here, keep doing labs, keep learning concepts, and keep improving. Great game development is in your reach.
